Ques 7.Explain the difference between JSON.stringify() and JSON.parse() using example.

Ques 7.Write the example to parse any JSON string in javascript?

Ans: JSON.stringify() and JSON.parse() are javascript method which we used for JSON data

Stringify

The JSON.stringify() method converts a JavaScript object or value to a JSON string.

const employee = {
firstName: 'Sourav',
state: 'Noida',
title:'React Developer'
};
const employeeString = JSON.stringify(employee);
console.log(employeeString);
Output: {"firstName":"Sourav","state":"Noida","title":"React Developer"}

Parse

The JSON.parse() method parses a JSON string and converts it into a JavaScript object or value.

const employeeString = '{"firstName":"Sourav","state":Noida,"title":"React Developer"}';
const employee = JSON.parse(employeeString);
console.log(employee);
Output: { firstName: 'Sourav', state: 'Noida', title: 'React Developer' }
